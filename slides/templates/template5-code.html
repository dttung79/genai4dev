Python GenAI Chat Implementation

  • Create Chat Session: Set up chat with Gemini 2.0 Flash model for conversation
  • Configure System Prompt: Define AI behavior with specific instructions and constraints
  • Interactive Loop: Handle user input and generate AI responses continuously
  • Exit Handling: Provide clean exit options with 'exit' or 'quit' commands
  • Response Display: Format and present AI responses with clear user/AI labels
chat_example.py Python 
client = genai.Client(api_key=get_key())
chat = client.chats.create(model='gemini-2.0-flash')

chat.send_message(message="You are a cat. You will answer questions as a cat. Convert words to meow, meoow, meooww, etc. based on the length of the word. ONLY MEOWS ALLOWED. No matter what the question is, you must respond with meows only.")

while True:
    question = input('[User]: ')
    if question == 'exit' or question == 'quit':
        break
    response = chat.send_message(message=question)
    print('[GenAI]:', response.text)
